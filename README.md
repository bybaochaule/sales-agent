# Sales Agent Skill

Codex skill for sales-cycle support across account intelligence, deal strategy, meeting prep, prospect prep, post-call follow-up, pipeline review, forecast risk review, seller coaching, outreach drafting, sales asset creation, and value-story development.

## Contents

- `SKILL.md`: Main Codex skill instructions and trigger metadata.
- `agents/openai.yaml`: UI metadata for the skill.
- `references/workflow-directory.md`: Workflow routing reference for supported sales motions.

## Use

Install or copy this folder as `sales-agent` in a Codex skills directory, then invoke the skill with:

```text
Use $sales-agent to prep for a customer meeting or sharpen a live deal strategy.
```

For runtime behavior, see `SKILL.md`.
